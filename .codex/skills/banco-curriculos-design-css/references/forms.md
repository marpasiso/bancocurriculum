# Form Guidelines

- Use visible labels for every input, select, and textarea.
- Keep each label/input pair vertically stacked unless the screen is clearly desktop-only.
- Use native input types: `email`, `password`, `number`, `checkbox`, and `select` where appropriate.
- Keep primary submit buttons full-width inside narrow forms.
- Put helper text near the relevant field.
- Do not rely on placeholder text as the only instruction.
- Keep required LGPD consent visually explicit and easy to read.
- Use server validation for trust; client attributes are only convenience.
