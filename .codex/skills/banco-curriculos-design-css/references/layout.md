# Layout Guidelines

- Keep `main` constrained with comfortable side padding.
- Use grid layouts with `repeat(auto-fit, minmax(...))` for responsive cards.
- Keep navigation simple and wrap links on small screens.
- Avoid nested cards.
- Use tables only when comparison is the main task; otherwise use cards or definition-like rows.
- Admin and employer pages should favor compact information density.
- Public candidate/LGPD pages should favor calm reading and clear completion.
- Avoid landing-page hero sections unless the user explicitly asks for a marketing page.
